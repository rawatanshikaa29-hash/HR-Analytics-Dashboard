# HR Analytics Dashboard

## Project Overview
This project analyzes HR employee attrition data using Excel, SQL, Python, and Power BI. The objective is to identify factors affecting employee attrition and provide interactive business insights.

## Tools Used
- Power BI
- SQL
- Python
- Microsoft Excel

## Project Files
- HR_Analytics_Dashboard.pbix
- HR_Employee_Data.xlsx
- HR_Attrition_Analysis.sql
- attrition.python

## Features
- Employee Attrition Analysis
- Department-wise Insights
- Gender Analysis
- Age Group Analysis
- Interactive Dashboard

## Skills Demonstrated
- Data Cleaning
- SQL Queries
- Data Analysis
- Data Visualization
- Dashboard Design
